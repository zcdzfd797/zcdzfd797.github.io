git

发表于 2019-05-04 | 分类于 工具 | | 阅读数

1、正式项目都需要代码版本管理
2、大型项目需要多人协作开发
3、Git和Linux是一个作者
4、一般公司代码非开源，都有自己的Git服务器

常用Git命令

  • git init — 本地库初始化
  • git status — 查看当前被git管理的文件
  • git add . — 添加操作
  • git commit -m "xxx" — 提交操作
  • git log — 查看日志信息
  • git push origin master — 将当前分支推送到origin主机的对应分支。
  • git pull origin master — 拉取代码到本地

分支操作

  • git branch — 查看分支
  • git branch xxx — 创建分支
  • git checkout -b xxx — 创建并切换分支
  • git merge xxx — 合并分支
  • git branch -d xxx — 删除master下的分支
