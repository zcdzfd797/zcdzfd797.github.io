slot 插槽

  • slot是Vue中内置组件— 凡是内置组件都不会被当做标签渲染到页面上
  • 优势： 当封装组件时，可以使组件更加灵活 ，复用性强
用处
  • 解决html页面文件 组件中写内容不显示问题
    • 在相应组件内写入 <slot>这里是默认显示值可以不写</slot>
    • 这时候在html文件 组件标签下可以写死文本了，不写则显示 slot中的默认数据
  • 有名字的 slot
    • html页面组件 <组件 slot="slot插槽名"> </组件>
    • js template模版

以此可以间接实现 父组件向子组件传递信息的途径
