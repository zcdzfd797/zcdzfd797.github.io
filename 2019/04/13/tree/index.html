前端工程师考试题

一、选择题

01、计算机中所有的信息采用的储存方式是：（ ）

A: 二进制 B： 十进制 C: 十六进制 D：ASCII码

02、用 MIPS 来衡量的计传输计算机性能指标是： （ ）

A: 传输速率 B： 存储容量 C: 字长 D：运算速度

03、十进制数 100 转换成二进制是： （ ）

A: 01100100 B： 01100101 C: 01100110 D：01101000

04、为了把一个汇编源程序或一个高级语言源程序转成可执行文件，需要一个相应的（ ）

A: 语言处理程序 B： windows系统 C: 操作系统 D：目标程序

05、设初始栈为空，s表示入栈操作，x表示出栈操作，则__ 是合法的操作程序（ ）

A: sxxsssxxx B： xxssxxss C: sxsxssxx D：xsssxxx

06、URL 的含义是： （ ）

A: 传输控制协议 B： Internet协议 C: 简单邮件传输协议 D：统一资源定位器

07、数据中的树最合适用来表示__ 的情况（ ）

A: 数据元素有序 B： 数据元素之间具有多对多关系 C: 数据元素无序 D：数据元素之间具有一对多关系

08、局域网常用的拓扑结构有：（ ）

A: 星形、总线型、环形、立方形 B： 星形、总线型、环形、广播形 C: 星形、总线型、环形、树形 D：星形、总线型、环形、螺旋形

09、202.202.32.20是什么类型的IP地址 （ ）

A: A类 B： B类 C: C类 D：D类

10、计算机中的最小信息单位是（ ）

A: Byte B： Bit C: KB D：GB

11、UML 是一种___ ( )

A: 面向对象的程序设计语言 B： 面向过程的程序设计语言 C: 软件系统开发方法 D：软件系统建模语言

12、采用哈希（或者散列）技术构造查询表时，需要考虑冲突（碰撞）的处理，冲突是指（ ）

  • A: 关键字相同的记录被映射到不同的哈希地址
  • B: 关键字不同的记录被映射到同一个哈希地址
  • C: 关键字依次被映射到编号连续的哈希地址
  • D: 关键字的数目超过哈希地址的数目

13、电子邮件是Internet应用最广泛的服务项目，通常采用的传输协议是（ ）

A: SMTP B： TCP/IP C: CSMA/CD D：IPX/SPX

14、计算机的存储单元中存储的内容：（ ）

A: 只能是数据 B： 只能是程序 C: 可以是数据和指令 D：只能是指令

15、在因特网体协结构中，属于网络层协议的有：（ ）

A: IP,TCP和UDP B： FTP,DNS,SMTP C: IP,ARP,RARP,ICMP D：IP,ARP,SNMP,POP3

16、WWW 的英文全称为： （ ）

A: World Wide World B： Work World Web C: World Web Wide D：World Wide Web

17、计算机病毒可以使整个计算机瘫痪危害极大。计算机病毒（ ）

A: 一条命令 B： 一段特殊的程序 C: 一种生物病毒 D：一种芯片

18、计算机的内存储器比外存储器：（ ）

A: 更便宜 B： 存取速度更快 C: 容量大 D：价格更便宜但速度慢

19、SQL语句的核心功能是：（ ）

A: 数据定义 B： 数据查询 C: 数据控制 D：数据操纵

20、安全的web服务器与客户机之间通过__协议进行通信。（ ）

A: HTTP+SSL B： TeInet+SSL C: TeInet+HTTP D：HTTP+FTP

21、如果a===a为false,则说明a是：（ ）

A: Infinity B： NaN C: Undefined D：Null

22、页面文档完全加载并解析完毕之后会触发的事件是：（ ）

A: ready B： load C: DOMContentLoaded D：DOMContentReady

23、'string' instanceof String 的结果是： （ ）

A: ture B： false

24、下面哪条语句可以获取元素e相邻的下一个元素（ ）

A: e.nextSibling() B： e.nextElement() C: e.nextElementSibling D：e.next

25、以下哪个单词不属于javascript关键字：（ ）

A: with B： parent C: class D：void

26、以下会触发reflow的操作有（多选）：（ ）

A: 改变任意元素height/width B： 改变background-color C: 改变浏览器窗口的大小 D：改变font-size

27、以下哪些可以作为`media queries的条件 ？（多选） （ ）

A: 设备宽度 B： 设备高度 C: 设备类型 D：设备像素比

28、下列说法正确的是：（多选）（ ）

  • A: 样式文件的加载会阻塞脚本的执行
  • B: iframe 会阻塞主页面的 load 事件
  • C: window.location.href='http://ele.me'和window.location.replace("http://ele.me")的效果是等价的
  • D: <audio>、<img>中的资源下载会阻塞页面解析

29、以下几组标签全部符合HTML5标准的是： （ ）

A: <section><audio><video> B： <address><svg><aside> C: <nav><image><footer> D：<fieldset><canvas><pre>

30、下列说法错误的是：（ ）

  • A: window.alert是一个同步方法，会阻塞脚本的执行
  • B: setTimeout(foo,0)这行代码可以等价替换为 foo()
  • C: 在addEventListener的处理方法中使用e.preventDefault()可以阻止事件冒泡
  • D: String('abc')和new String('abc') 是等价的

31、以下代码的运行结果是：（ ）

A: t B： r C: '' D：string

32、下列属于块级元素的是：（多选） （ ）

A: <p> B： <span> C: <h3> D：<n>

33、下列关于css的描述，错误的是：（多选）（ ）

  • A: 内联样式声明拥有最高优先级
  • B: 缺省情况下都是块级元素
  • C: position:absolute的元素总是相对于position:relative的元素定位
  • D: cm em px pt 均属于CSS尺寸单位

34、在javascript里，下列选项中不属于数组方法的是（ ）

A: sort() B： length() C: concat() D：reverse()

35、下列选项中，描述正确的是（多选）（ ）

  • A: options.add(new Option("a","A")) 可以动态添加一个下拉列表选项
  • B: option.add(new Option("a","A")) 可以动态添加一个下拉列表选项
  • C: new Option("a","A") 中的 “a” 表示列表选项的值。”A”用于在页面中显示
  • D: new Option("a","A") 中的 “A” 表示列表选项的值。”a”用于在页面中显示
