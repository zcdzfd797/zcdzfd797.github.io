Hexo博客建站被坑指南

发表于 2019-04-03 | 分类于 Hexo | | 阅读数

Hexo

使用Hexo的理由

  • Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。最主要是它能配合github使用，可以直接浏览md文档，并与他人分享。

    安装前提

  • Node.js
  • Git

    安装

  • 全局安装Hexo后，选定一个位置创建目录放置文件，如（D:\blog）
  • 进入这个文件夹，cmd

运行

  • 浏览器输入 localhost:端口号

博客发布到github

  1. 打开 _config.yml

部署好之后，我们就可以上传我们的代码
回到命令行窗口，输入

再输入hexo g，
然后再hexo d，就可以将我们public里面的代码上传咯，在GitHub上可以看到我们上传的代码。这样别人也可以通过域名访问我们博客了。在地址栏输入http://域名就可以访问。

坑1：更改主题 要 hexo clean

创建github.io网站

1）点击New repository
2）输入Repository name，必需为username.github.io格式。username替换为用户名
3）点击按钮Create repository
4）进入仓库username.github.io,点击setting,找到GitHub Pages模块
5）点击choose a theme选择一个页面主题
6）访问https://username.github.io就可以访问博客网站了

载入图片问题

  • 首先把blog（hexo）目录下的_config.yml里的psot_asset_folder:设置为true

  • 安装

  • 运行hexo new 博客名 来生成md博客时，会在_post目录下看到一个与博客同名的文件夹

  • 将想要上传的图片先扔到文件夹下，然后在博客中使用markdown的格式引入图片
