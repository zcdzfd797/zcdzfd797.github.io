用途

  • flexible 会为页面根据屏幕自动添加 <meta name='viewport' > 标签，动态控制 initial-scale，maximum-scale，minimum-scale 等属性的值。
  • postcss-px2rem会将px转换为rem，rem单位用于适配不同宽度的屏幕，根据标签的font-size值来计算出结果，1rem=html标签的font-size值。

配置

  1. 移除网页的标签
    • 移除 目录 public/index.html 中的 <meta></meta>标签。（flexible会自动添加）

  2. 引入lib-flexible

    • 在项目入口文件 main.js中引入lib-flexible

  3. 配置postcss-px2rem

    • 打开根目录下的 vue.config.js ,添加

备注

  • 如果个别地方不想转化px。可以简单的使用大写的 PX 或 Px 。
  • 插件会转化所有的样式的px。比如引入了三方UI，也会被转化。
  • cli3没有该vue.config.js 文件需要手动创建。
