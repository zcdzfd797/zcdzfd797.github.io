cordova制作app

发表于 2019-03-11 | 分类于 web前端 | | 阅读数

APP

app类别

  • webapp — 浏览器端
  • hybird — 移动端app 将ios/android 变成浏览器（开发的语句还是原生+html+css）
  • nativeApp — vue/react/angular –> 原生的ios/android

    可以替你形成app的(古老稳定的)框架：

  • ionic — 最活跃,暂仅支持angular
  • cordova
  • phoneGap

    vue 对应 WEEX
    react 对应 REACT-NATIVE [2019年1月1日 风靡 ]
    angular 对应 nativeSctipt

app功能列表示例

  • 获取地理位置
  • 获取当前手机设备信息
  • 扫描
  • 上传头像、图片
  • 拍照
  • 支付 — 调用第三方接口

cordova 框架的学习

  • cordova的用处，生成能在手机app上运行的壳

    cordova 启动流程

  • 官网链接
  • 全局安装 npm i -g cordova
  • 查看是否安装成功 cordova -v

    出现bug提示 cordova不是内部或外部命令

    解决方案：重新配置环境变量C:\Users\Administrator\AppData\Roaming\npm填入[系统-高级系统设置–环境变量—填写]

  • 创建项目

    cordova create 项目名称
    cordova create .com公司名称.项目名 [不建议这么写，原因不明]

    文件构成

    • | hooks —> 勿删
    • | platforms —> 要打包成什么平台 ios /android
    • | plugins —> 扩充功能 如 sanner divices bettersalary …
    • | res —> 当前应用的配置资源（eg:不同清晰度的图标）
    • | www —> 前端资源放置位置
    • .npmignore — 忽略你在发布包时不想发布的文件
    • config.xml — 配置项目名称等

cordova 的使用

环境配置

  • ios —- xcode(从苹果的 app store 注:”超级超级慢” )
  • android —- 配置[java(安装jdk Java SE Development Kit,并添加Java环境变量配置) + 安装 android.studio]

    提前层级不深的位置准备一个文件夹，如D盘 andriod文件夹 此时会有 sdk :安卓配置包 studio：android.studio工具。建议使用64位的包

android studio 环境搭建

此部分需百度。…… 以上大约耗时1上午

开发
  • 添加平台： cordova platform add android/ios —约耗时20分钟+ （待考证）

    新增内容： node_modules*** 内有插件、platforms - android

    打包
    • 打开 android studio 应用
    • 从应用中打开创建的项目文件夹内平台下的 android /ios 文件夹
  • 运行代码： cordova run android

    android studio 应用 点击左上角 的 小锤子（第七个图标）

  • 安装app：需先连接设备（设备需要打开开发者模式–>调试模式-这里自行百度-）.

    若为提示连接成功 cmd adb devices —显示设备唯一标识号
    手机会多一个app，其内容就是www文件夹内容

  • 更新app： cordova prepare +应用的绿色开始按钮▷

插件

  • 插件安装 cordova plugin add 插件名 上万种插件plugin
  • 查看已安装插件 cordova plugin list
    设备信息 cordova-plugin-device
  • 安装 cordova plugin add cordova-plugin-device

    安装此device插件后 全局window上有一个对象：device。拥有很多属性

    • device.cordova — cordova版本
    • device.model — 底层模型
    • device.platform — 平台
    • device.uuid — 手机唯一标识
    • device.version — 版本号
    • device.manufacturer
    • device.isVirtual
    • device.serial — 系列

相关API使用复制粘贴官方文档

app打包 vue版

  • 修改 config — index.js—build: { index路径更改至www，assetsRout 路径更改至www assetsPublicPath:'./'}

  • 在app的index 引入<script type="text/javascript" src="cordova.js"></script>

  • 可使用vue插件 vue-cordova

  • weex-ui | natjs
